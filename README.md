# NoQ 🚀

## 📌 Problem
In college hostels, dining halls often get overcrowded.  
Students go to eat, see long queues, and return without food.

---

## 💡 Solution
**NoQ** is a smart Android app that helps students avoid crowds by showing **real-time dining hall status**.

---

## ⚙️ Features
- 📹 Live CCTV-based crowd monitoring  
- 📊 Real-time crowd percentage & density  
- 🔔 Smart notifications (empty / crowded alerts)  
- 🔐 Secure login with college email only  
- 📧 OTP verification for authentication  

---

## 🛠️ Tech Stack
- React Native (Expo)  
- Node.js + Express  
- MongoDB  
- Python (for processing)  
- Expo Push Notifications  

---

## 🚀 How it Works
- Fetches live CCTV data from dining halls  
- Processes crowd density  
- Displays real-time status in the app  
- Sends alerts when halls are empty or crowded  

---

## 🎯 Goal
To help students save time, avoid queues, and never miss meals.
